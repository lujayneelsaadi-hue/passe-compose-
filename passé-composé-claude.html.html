01 — Formation

Comment former le passé composé ?

Formule : Auxiliaire au présent + Participe passé du verbe
L'auxiliaire est soit être, soit avoir conjugué au présent de l'indicatif.

Quel auxiliaire choisir ?
AVOIR La majorité des verbes ▾

La grande majorité des verbes se conjuguent avec avoir.

–
J'ai mangé une pomme.
manger → avoir → mangé (1er groupe, -é)
–
Ils ont fini le travail.
finir → avoir → fini (2e groupe, -i)
–
Elle a pris le train.
prendre → avoir → pris (3e groupe, irrégulier)
ÊTRE Verbes de mouvement + pronominaux ▾

Les verbes de mouvement et les verbes pronominaux utilisent être.

–
Elle est arrivée en retard.
arriver → être → accord féminin : -ée
–
Ils sont partis.
partir → être → accord masculin pluriel : -s
–
Elle s'est lavée.
se laver → pronominal → accord sujet féminin : -ée

Verbes de mouvement courants : aller, venir, entrer, sortir, partir, arriver, naître, mourir, monter, descendre, passer, rester, retourner, tomber…

02 — Accords

L'accord du participe passé

ÊTRE — Accord avec le sujet
Avec être, le participe passé s'accorde toujours en genre et en nombre avec le sujet.
✓
Elle est partie.
sujet : elle (féminin singulier) → +e
✓
Ils sont arrivés.
sujet : ils (masculin pluriel) → +s
✓
Nous sommes allées.
sujet : nous (féminin pluriel, ex: groupe de filles) → +es
AVOIR — Accord avec le COD antéposé
Avec avoir, le participe passé est invariable en général.
Exception : accord si le COD (complément d'objet direct) est placé avant le verbe.
✗
J'ai mangé des pommes.
COD « des pommes » est APRÈS → pas d'accord
✓
Les pommes que j'ai mangées.
COD « les pommes » (féminin pluriel) est AVANT → accord : -ées
✓
La lettre qu'il a écrite.
COD « la lettre » (féminin singulier) est AVANT → accord : -e
03 — Exceptions & Pièges

Les cas difficiles (niveau Bac)

COD vs COI — La distinction essentielle
COD → peut déclencher l'accord
Question : quoi ? qui ?
Pronoms : le, la, les, l'
Pronom relatif : que
COI → jamais d'accord
Question : à qui ? de quoi ?
Pronoms : lui, leur, y, en
Pronom relatif : dont
✓
Les fleurs que j'ai achetées.
« que » remplace « les fleurs » → COD → accord féminin pluriel
✗
Les fleurs dont j'ai parlé (invariable).
« dont » = COI (parler DE) → jamais d'accord

Verbes pronominaux — Le grand piège du Bac
Tous les verbes pronominaux utilisent être, mais l'accord dépend du rôle du pronom réfléchi.
Cas 1 : Le pronom est COD → ACCORD ▾
✓
Elle s'est lavée.
Elle a lavé QUI ? → « se » = COD → accord avec « elle » → -ée
✓
Elles se sont vues hier.
Elles ont vu QUI ? → « se » = COD → accord féminin pluriel → -es
Cas 2 : Le pronom est COI → PAS d'accord ▾
✗
Elle s'est lavé (invariable) les mains.
Elle a lavé QUOI ? → « les mains » = COD / « se » = COI → pas d'accord
Cas spécial : se rendre compte → toujours invariable ▾
« compte » = COD · « se » = COI → donc jamais d'accord
✓
Elle s'est rendu (invariable) compte de son erreur.
✓
Elles se sont rendu (invariable) compte du problème.
Verbes avec ÊTRE mais qui peuvent utiliser AVOIR
Certains verbes changent de sens selon l'auxiliaire !
Avec ÊTREAvec AVOIR
Il est sorti. → (il est dehors)Il a sorti la poubelle. → (COD présent)
Elle est montée. → (elle est en haut)Elle a monté les valises. → (action transitive)
Ils sont descendus. → (ils sont en bas)Ils ont descendu les cartons. → (COD présent)
